# BookZy

Appointment booking SaaS for Indian small service businesses.

## MVP

BookZy gives each business a public booking link where customers can choose a service, pick an available slot, and confirm an appointment without creating an account. Business owners manage services, working hours, bookings, blocked slots, and no-shows from a dashboard.

## Tech Stack

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Supabase Auth, PostgreSQL, RLS, and Storage
- Resend for email confirmations
- WhatsApp provider integration planned after core booking flow
- Vercel deployment
